## What

Small changes that compound: typography, spacing rhythm, and transitions

## Why

This is where "nice app" becomes "stunning enough to share."

## How

- Use a consistent type scale:
    - Page title: `text-2xl font-semibold`
    - Section headers: `text-lg font-semibold`
    - Body: `text-sm/relaxed`
- Add sublte transitions:
    - Chart switch: fade/slide 150-200ms
    - Results appear: fade-in
- Ensure empty state is not too empty: small preview skeleton.

## Success criteria

- No awkward whitespace “dead zones.”
- Page feels smooth, not jumpy.
